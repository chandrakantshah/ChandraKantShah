## ⚡ Key Feature (Data-Driven Architecture)

This portfolio is **fully dynamic**:

- Content stored in `portfolio.xml`
- JavaScript fetches and renders content
- Easy updates without touching UI code

👉 Just edit XML → website updates automatically
